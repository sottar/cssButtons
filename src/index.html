CSS Buttons

HTMLとCSSのみで作れるボタン集

CSSのみで作成できるインタラクティブなボタンデザインをまとめました。使いやすいものや面白いものをまとめてます。
対象ブラウザに古いIEなどが含まれている場合は使えませんが、いずれもデザインやアニメーションを全てCSSのみで実装しているものです。
コードをコピペするだけで使えるようになってますのでお試しください。

ボタンをクリックするとそのボタンのHTMLとCSSが表示されます。

ボタンのサイズや色などは各自で変更してください。
